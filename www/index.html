Project Darkstar C Client
The sgs-c-client is a Project Darkstar C client API that can be used to interact with an application running on top of a Project Darkstar server. This site provides access to the source code repository, issue tracker, and mailing lists for this project. Please see http://www.projectdarkstar.com for more information.
Source code
The publicly accessible Subversion repository contains all of the source code for the sgs-c-client. In addition to the java.net Subversion browser linked above, the repository may also be viewed using the FishEye browser.
Issue tracker
Issuezilla is used to track all issues with the sgs-c-client including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the sgs-c-client. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
